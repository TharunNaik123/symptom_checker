# Healthcare Symptom Checker
#  Symptom Checker System

## Overview
This project is a simple healthcare symptom checker that takes user input symptoms and provides possible conditions along with recommendations.
##  Description

## Features
- Input symptoms
- AI-generated conditions
- Suggested next steps
- Medical disclaimer
- Fully offline using Ollama
I built this project to understand how machine learning can be used in healthcare.
It takes user symptoms as input and predicts possible diseases based on the data.

## Tech Stack
- FastAPI (Backend)
- HTML, JavaScript (Frontend)
- Ollama (Local LLM)
---

## How to Run
##  Features

1. Install dependencies:
pip install fastapi uvicorn requests
* User can enter symptoms
* System predicts possible disease
* Simple and easy-to-use interface
* Fast response

2. Run Ollama:
ollama run llama3
---

3. Start backend:
uvicorn main:app --reload
##  Tech Stack

4. Open index.html in browser
* Python
* Machine Learning
* Pandas / NumPy
* (Flask / Streamlit if used)
